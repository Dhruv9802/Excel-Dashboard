Employee Salary Dashboard (Excel)

This folder contains an Employee Salary Dashboard created using Microsoft Excel to analyze salary distribution and workforce insights across different departments.

Dashboard Highlights
- Total salary and average salary overview
- Department-wise salary comparison
- Top 5 highest-paid employees
- Salary contribution by department
- Employee joining trend over the years

Key KPIs
- Total Salary
- Average Salary
- Active Employees Count
- Top Salary
- Department-wise Scores

Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Slicers for department-level filtering
